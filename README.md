kernel
======

Kernel for the beagleboard.org boards

usage
======

beagleboard-3.2 patchset:

git clone git://github.com/beagleboard/kernel.git

cd kernel

git checkout origin/beagleboard-3.2 -b beagleboard-3.2

./patch.sh
